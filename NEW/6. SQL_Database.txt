SQL
---
SQL : Structured Query Language
Query language is used to interact with Relatiional Databases

1.Core SQL Operations
----------------------
CRUD - Create : Create new Tables,indexes and views
       Read   : Retrieve data from one or more tables
       Update : Modify existing data within a table
       Delete : Remove data from a table


2. Database
------------
  Data     : any information stored in computer memmory
  Database : systemic collection of data which can be stored ,accessed,manipulated,updated and deleted

3. TYPES OF DATABASE
----------------------
  Relational Database (RDBMS)  : eg: sqlite,mysql,postgresql,oracle db,ms sql server
  Hierarchical Database        : IBM 
  Network Database             : IDS
  Object-oriented Database     : db40,ObjectDB
  NoSQL Database               : mongodb,couchdb
  NewSQL Database              : google spanner,cockroach DB
  Cloud Database               : amazon RDS,Google cloud SQL, Azure SQL Database
  In Memmory Database          : Redis, Memcached

4. Structure
------------
  collection - table (data organized in rows and columns)
  record     - rows (record/tuple)
  fields     - columns (attribute)

  PRIMARY KEY : field in table which uniquely identifies each row
  FOREIGN KEY : a feield which refers primary key of another table
