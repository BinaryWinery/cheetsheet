FILESYSTEM
----------
/ - root file (contains /home , /etc , /var , /tmp , /boot , /bin subfolders)

/home - contains the personal home directories for regular ,non-root users
/etc  - contains system-wide configuration files for nearly all programs and services
/var  - Contains variable data. This data changes frequently during system operation
/tmp  - Contains temporary files. Files in /tmp are often deleted on system reboot
/boot - Contains essential files for system boot-up, including the Linux kernel
/bin  - Contains essential command binaries for all users (including root) to operate the system, like ls, cat, and mv


COMMANDS
--------

df                : shows total/used/free/location of each drive
free              : shows the free RAM memmory (physical) and swap memmory 
date              : shows date and time 
touch [filename]  : creates a file with filename provided
