1. Initialize Module
___________________________
	>> go mod init [name]
	(eg) >>  go mod init myTest
2. Extension
-------------
	filename.go

3. Print Hello world
---------------------
	package main
	
	import "fmt"
	
	func main() {
		fmt.Println("Hello world")
	}
result : 
	Hello world

4. Run
-------
	>> go run filename.go
	(eg) >> go run main.go
