1. Initialize Module
___________________________
	>> go mod init [name]
	(eg) >>  go mod init myTest
2. Extension
-------------
	[filename].go
	main.go

3. Print Hello world
---------------------
	package main
	
	import "fmt"
	
	func main() {
		fmt.Println("Hello world")
	}

----------------------------------
	Hello World

4. Run or Build
----------------
	Run:
	>> go run [filename].go
	(eg) >> go run main.go

	Build:
	>>  go build [filename].go
	(eg) >> go build main.go
